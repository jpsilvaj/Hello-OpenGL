1 - Lista de comandos:

C/c - Muda a cor do objeto em uma escala predeterminada.
B/b - Muda a cor do background em uma escala predeterminada.
Q - Muda a primitiva para um quadrado.
T - Muda a primitiva para um triângulo.
P - Muda a primitiva para um polígono.
ESC - Sai da aplicação.
Espaço - Reinicia o sistema.

2 - Botões:

Botão "Escalamento +" - Aumento o fator de escala do objeto em um valor pré-definido.
Botão "Escalamento -" - Diminui o fator de escala do objeto em um valor pré-definido.
Botão "Rotacionamento H" - Rotaciona o objeto no sentido horário.
Botão "Rotacionamento AH" - Rotaciona o obejto no sentido anti-horário.
